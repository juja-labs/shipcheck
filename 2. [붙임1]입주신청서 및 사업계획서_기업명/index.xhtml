2. 작성요령

- 사업계획서는 최대 10페이지를 넘지 않도록 작성(신청서, 표지, 목차 제외)

- 해당사항 없을 시 작성하지 않을 수 있으나, 서면평가에 반영될 수 있음

- 각 칸의 부연설명 글씨는 모두 삭제 후 PDF로 변환하여 업로드

3. 추가 서류가 있는 경우 첨부 가능 (예, 기술소개서/회사소개서 등)

4. 제출된 자료는 반환되지 않음
