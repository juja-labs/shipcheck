- 소재지: 서울시 마포구 서울창업허브 본관 513
- 주요 사업 모델:
  - B2C: 주차 서비스 매칭 플랫폼 "잇차"

### 2. Market

**글로벌 스마트 주차 시장**
- 2018년 $1.9B → 2023년 $3.8B (CAGR 14%)
- 출처: IoT Analytics "Smart Parking Report 2019-2023"

**국내 주차 시장**
- 2020년 추산 시장규모: 약 5조원 (카카오 모빌리티 추산)

**발렛 시장 규모**
- 강남구 기준: 약 1,920억원 추정
  - 7개 업체 계약, 월매출 2,500~3,000만원 → 1매장 월 400만원
  - 1,000개 발렛 사업체 × 4매장 × 400만원 × 12개월
- 서울 전체: 약 5,000여개 매장 → 2,400억 규모

**TAM/SAM/SOM**
- TAM: 약 18.8조원 (전체 유료 주차면 8,700만면)
- SAM: 5.5조원 (수도권 유료 주차면 4,200만면)
- SOM: 1.54조원 (서울 주요 유료 주차면 568,000 + 강남 발렛)
- 목표: SOM의 30%

### 3. Pain Point

**주차 문제의 본질**
- 서울시 부족 주차면: 240만면 (여의도 10개 면적)
- 주차장 확보율: 136% (수요의 77% 부족)
- 주차장 확보 총 예산: 192조원
- 핵심: 공간과 시간의 수요 불균형과 공급 비대칭

**주차시장 구조적 문제**
- 공급(부동산 소유주) → 운영(주차사업 인프라) → 수요(운전자) 밸류체인이 복잡
- 공급자: 복잡한 구조와 높은 운영비용 → 적자 구조
- 수요자: 분절된 부가서비스 → 시간과 비용 낭비
- 운영 발렛사업주: 매출 불투명, 인력 관리 한계

**발렛시장 고유 문제**
- 불신: 책임배상보험/탁송보험 적용 불가, 카드결제 불가
- 불투명: 현금 운영, 시장규모 파악 어려움
- 서비스 표준화 부재

### 4. Business Model

**B2C "잇차" — 주차를 시작으로 하는 차량 UX 개선 플랫폼**
- 플로우: 이동 목적지 설정 → 드라이버 링커 매칭 → 서비스 수행 → 출차요청 → 결제/평가
- 핵심: 주차장을 찾거나 예약할 필요 없이 목적지 앞에서 주차대행
- 전략 Step: 유휴주차면 활용 → 서비스 기반(링커 운영) → 플랫폼(수요공급 통합)

**B2B "링커" — 주차대행 서비스 현장 운영 관리 플랫폼**
- 플로우: 링커 인증(GPS) → 입차 → 출차 → 결제 → 운행 데이터 제공
- 핵심: 발렛사업주의 인력/차량/회계/결제를 일괄 관리
- 전략 Step: 플랫폼 D/T 수행 → 운영 Data 제공 → Data 기반 직접 운영권 획득

## IR Deck 구조 패턴 분석 (Personica 작성 시 참고)

### 전체 흐름 공식
```
Cover → 회사 미션 → Company Overview → Market → Pain Point → Business Model → Traction/Reference → Business Plan → Team → Finance Plan → Appendix
```

### 주목할 패턴

1. **섹션 구분 슬라이드**: 각 대섹션 앞에 다크 배경 + 금색 테두리의 섹션명 슬라이드 사용 → 발표 시 흐름 전환 명확
2. **문제 → 솔루션 순서**: Market(기회) → Pain Point(문제) → Business Model(솔루션) 순서로 배치
3. **B2C/B2B 병렬 구조**: 두 사업 모델을 항상 좌우 병렬로 비교 (문제 정의, 제품, 매출 구조 모두)
4. **Unit Economics 조기 제시**: Business Model 섹션에서 바로 제품별 매출 구조/마진율 제시
5. **Traction 데이터**: 분기별 추이 차트 + 절대 수치 동시 제시
6. **사업 계획 4단계 스텝**: Step1~4로 시간순 확장 로드맵을 시각적으로 전개
7. **자금 활용 구체적 배분**: 투자금의 %별 사용처 명시 (인수 51%, 운전 34%, R&D 15%)
8. **팀 소개 방식**: C-Level(사진+약력) → Advisor(전문분야별) → Product Team(전원 사진+기술스택)
9. **TAM/SAM/SOM**: 공신력 있는 출처 명시 + Bottom-up 추정 로직 별도 슬라이드
10. **협업/투자 이력 로고 월**: 제휴사/투자사/수상 로고를 한 슬라이드에 격자형 배치
