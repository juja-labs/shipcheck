Executive Summary

Tally.so의 핵심 폼 빌더 경험은 대다수 페르소나에게 긍정적으로 평가되었으나, 템플릿 탐색 실패 시 복구 경로의 부재가 이탈의 주요 원인으로 나타났습니다. 조건 분기 기능은 고급 사용자에게 높은 가치를 제공하지만, 설정 과정의 인지 부하가 진입 장벽으로 작용합니다. 공유 플로우의 마지막 단계에서 명확한 완료 피드백을 추가하면 전환율 개선이 기대됩니다.

  • 템플릿 검색 결과가 없을 때 "빈 폼으로 시작하기" CTA와 관련 추천 표시
  • 조건 분기 설정에 단계별 가이드 또는 프리셋 템플릿 도입
  • 공유 링크 복사 시 애니메이션 피드백 및 "다음 단계" 안내 추가
