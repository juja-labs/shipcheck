핵심 발견 3가지

#1

전체 UI가 영어 → 비-IT 유저 즉시 이탈 위험

박서연은 첫 화면부터 "한국어가 하나도 없네"라고 반응. 템플릿 목록에서도 Registration form, Lead generation form 등 영어 용어에 막혀 "이게 다 뭔 말인지 하나도 모르겠다"며 패닉.

치명적
#2

핵심 기능(조건부 로직)이 블록 드롭다운에서 발견 불가

김도현은 '/' 드롭다운에서 조건부 로직을 찾으려 스크롤과 타이핑을 반복했지만 끝내 발견하지 못함. "검색 필터가 작동하는 건지 아닌 건지 피드백도 없다."

치명적
#3

'Sign up' 버튼이 불안 유발 — 가입 없이 쓸 수 있는 건지 혼란

두 페르소나 모두 우측 상단의 'Sign up' 버튼에 반복적으로 불안을 느낌. 박서연: "결국 가입을 해야 하는 건지 확인하고 싶은데, 가입하기는 싫고..." 김도현: "작업하다 날아가면 짜증날 것 같다."

나
Tally를 처음 열었을 때 첫인상이 어땠어요?
박서연
전부 영어더라고요. 'The simplest way to create forms'라고 써있었는데, 솔직히 구글 폼도 잘 쓰고 있는데 왜 이걸 써야 하나 싶었어요. 그나마 'No signup required'라는 게 보여서 가입 없이 써볼 수 있다길래 일단 들어가봤는데... 한국어가 하나도 없는 건 좀 당황스러웠어요.

템플릿 고르는 화면에서 어떤 점이 어려웠나요?
박서연
Registration form, Lead generation form, Web Project Intake Questionnaire... 이런 게 쭉 나오는데 하나도 모르겠더라고요. 학부모 설문이랑 비슷한 게 어딨는지 한참 찾다가 겨우 'Customer Satisfaction Survey'라는 걸 봤는데, 이게 '고객 만족도'지 학부모 설문이랑 같은 건지 확신이 안 들었어요. 한국어 템플릿이 하나라도 있었으면...

나
결국 왜 나가셨어요?
박서연
오른쪽 위에 'Sign up'이라는 파란 글씨가 계속 보이는데, 아까 'No signup required'라고 해놓고 결국 가입해야 쓸 수 있는 건가 싶었어요. 가입까지 해서 써야 할 정도로 필요한 건 아니고... 구글 폼이 한국어도 되고 익숙하니까 그냥 그걸로 하자 싶었어요. 새로운 거 배우는 게 쉽지 않은 나이다 보니요.
