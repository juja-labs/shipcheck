## 1. 문제인식 (Problem)

### 1-1. 창업아이템 배경 및 필요성

제품 출시, 시장 진입, 서비스 설계 — 모든 비즈니스 의사결정의 핵심은 **"다양한 사람들이 이것을 어떻게 받아들일까?"**이다. 하지만 실제로 다양한 사람의 관점에서 미리 확인할 방법은 여전히 비싸고, 느리고, 편향되어 있다.

| 영역 | 현재 방법 | 비용 | 한계 |
|------|----------|------|------|
| **제품 UX 검증** | 베타 테스터 모집, 지인 테스트 | $2,000~5,000 / 2주 | 편향 심함. 1인 창업자에겐 비현실적 |
| **사용자 리서치** | 인터뷰, 설문, FGI | $5,000~46,000/년 | 참여자 모집 어려움, 수 주 소요 |
| **전환 퍼널 최적화** | A/B 테스트, Amplitude | 4~6주 / 릴리즈 | 출시 후에만 가능, 사전 검증 불가 |
| **전문가 리뷰** | 외부 컨설턴트 | $200~500/시간 | 단일 관점, 비용 부담 |

> *"80명에게 직접 공유했는데, 겨우 1명이 가입했다. 나머지 79명은 왜 안 왔는지 물어볼 방법도 없었다."*
> — Indie Hackers 커뮤니티에서 반복적으로 등장하는 패턴

매년 15,000개 이상의 웹 제품이 출시된다. **만드는 건 쉬워졌지만, "사람들이 이걸 원하는가"를 확인하는 건 여전히 어렵다.** 사용자 리서치는 UX 테스트, 설문, 인터뷰, 퍼널 분석 등 여러 방법론으로 분산되어 있고, 각각 별도의 도구와 참여자가 필요하다. 하나의 플랫폼에서 모든 리서치를 빠르고 저렴하게 수행할 수 있다면?

**지금, AI가 사람처럼 행동하고 판단할 수 있는 기술이 성숙했다.**

| 기술 변화 | 시기 | 의미 |
|----------|------|------|
| LLM API 비용 전년 대비 80% 하락 | 2025~2026 | 수백 명 규모 시뮬레이션이 경제적으로 가능해짐 |
| Playwright, Browser Use 등 브라우저 자동화 성숙 | 2024~2025 | AI가 실제 웹 제품을 직접 클릭·입력·탐색 가능 |
| "합성 사용자" 카테고리에 $160M+ VC 투자 유입 | 2024~2026 | 시장이 형성되고 있다는 투자 검증 |

Personica는 이 기술 성숙 위에 **"성격과 감정을 가진 페르소나"**를 올린다. 단순히 AI에게 역할을 부여하는 것이 아니라, 인지심리학 기반의 감정·인지·의사결정 모델로 **사람처럼 반응하는 AI 페르소나가 모든 유형의 사용자 리서치를 수행하는 종합 플랫폼**이다.

### 1-2. 창업아이템 목표시장(고객) 현황 분석

| | **Simfolk** | **Personica** |
|---|---|---|
| **입력** | Figma / 스크린샷 / 텍스트 | 실제 URL + 테스트 계정 (+ 디자인 파일) |
| **AI가 하는 것** | 디자인을 보고 행동을 **추론** | 실제 브라우저에서 제품을 **사용** |
| **시뮬레이션 깊이** | 다중 에이전트 LLM (프롬프트 수준) | **5-Layer 심리 모델** (인지+감정+의사결정) |
| **웹사이트 테스트** | COMING SOON | **구현 완료** |
| **결과 근거** | "이 UI면 혼란스러워할 것" (추론) | "실제로 클릭했더니 3초 망설이다 이탈" (행동) |

> Simfolk를 포함한 모든 경쟁사는 디자인 파일·스크린샷 기반의 간접 추론을 한다. **AI가 실제 제품을 사람처럼 사용하면서 감정/인지/의사결정을 시뮬레이션하는 서비스는 Personica뿐이다.**

## 2. 실현가능성 (Solution)

### 2-1. 창업아이템 현황 (준비 정도)

**핵심 시뮬레이션 엔진 개발 및 초기 검증 완료** (2026.01~03).

5-Layer 시뮬레이션 아키텍처 중 핵심인 **성격 프로파일(Layer 1)과 감정 시뮬레이션(Layer 3)**이 코드로 구현되어 작동한다. 인지 모델(Layer 2), 의사결정(Layer 4), 기억(Layer 5)은 설계가 완료되었으며 단계적으로 구현 예정이다.

**확인된 초기 신호:**

| 항목 | 결과 | 다음 단계 |
|---|---|---|
| **페르소나 간 행동 차이** | 같은 제품이어도 성격에 따라 이탈 시점이 5스텝 차이, 이유도 완전히 다름 | 50명+ 규모 통계적 분화 검증 (ANOVA) |
| **감정 모델의 효과** | 감정 모델 없이는 AI가 제품에 관대해짐 — 유용성 점수 0.3에서 0.7로 부풀려짐 | 체계적 대조 실험 (n=10+) |
| **실제 리뷰와 방향 일치** | Tally.so G2 리뷰 88개의 주요 불만 테마를 시뮬레이션이 독립적으로 발견 | 대규모 정량 비교 실험 |
| **원가 구조** | API 비용 기반 검증. 모든 시나리오에서 매출 총이익률 78% 이상 | 모델 전환(GPT-4.1 mini) 후 실측 |

### 2-2. 창업아이템 실현 및 구체화 방안

**왜 ChatGPT에 "리뷰해줘"와 근본적으로 다른가:**

ChatGPT에게 "52세 교사인 척 리뷰해줘"라고 하면 "사용하기 편리하네요!"라고 답한다. 실제 52세 교사는 영어 화면에서 겁을 먹고 떠났는데도. AI는 본질적으로 긍정 편향(sycophancy)을 가지고 있다.

Personica는 **감정 시뮬레이션 엔진**으로 이 문제를 구조적으로 완화한다:

| 지표 | 감정 모델 있음 (Personica) | 없음 (ChatGPT 수준) |
|---|:---:|:---:|
| 제품 유용성 점수 | 0.3~0.5 (솔직) | **0.6~0.8** (관대) |
| "포기하고 싶다" 언급 | 4회 | **0회** |
| 매몰 비용 심리 자연 발생 | 있음 | 없음 |

> 감정 모델 없이는 AI가 항상 호의적이다. **솔직한 피드백이 아니라 칭찬만 받게 된다.**

**핵심 기술 — 5-Layer 시뮬레이션 엔진:**

Personica의 핵심 IP는 5-Layer 시뮬레이션 엔진이다. 매 상호작용마다 아래 파이프라인이 실행되어, AI가 성격과 감정에 따라 사람처럼 반응하도록 한다.

```
[Layer 1] 페르소나 성격 프로파일                              ← 구현 완료
  ↓  성격(Big Five) → 행동 파라미터 수학적 매핑 (클릭 망설임, 에러 허용도 등)
[Layer 2] 인지 평가                                         ← 설계 완료, 구현 예정
  ↓  UI 복잡도 추적, 정보 탐색 패턴, 기대 vs 실제 비교
[Layer 3] 감정 평가                                         ← 구현 완료
  ↓  상황 → 감정 분류(OCC) → 쾌·각성·지배감 연속 추적(PAD) → 확률적 감정 진화(SDE)
[Layer 4] 의사결정                                          ← 설계 완료, 구현 예정
  ↓  동기 × 능력 × 계기 = 행동 여부 판단
[Layer 5] 기억                                              ← 설계 완료, 구현 예정
  ↓  경험 축적, 반성, 습관 형성, 세션 간 일관성 유지
```

각 레이어는 최신 인지심리학 논문에 기반한다 — PRISM(Fudan, 2025)의 확률적 감정 진화, Generative Agents(Stanford, 2023)의 기억 스트림, Concordia(Google DeepMind)의 에이전트 아키텍처 등 30+편의 논문을 분석하여 설계했다.

**종합 리서치 플랫폼 — 8개 리서치 유형:**

이 엔진은 특정 용도에 묶이지 않는다. 하나의 플랫폼에서 모든 사용자 리서치를 수행할 수 있다:

| 리서치 유형 | 설명 | 현재 상태 |
|---|---|---|
| **① 웹사이트 테스트** | AI 페르소나가 **실제 브라우저에서** 제품을 사용하고 행동/감정 로그를 수집. **경쟁사 전원 미보유 — Personica 유일의 핵심 차별화** | **엔진 개발 완료, 초기 검증 중** |
| **② UI/UX 디자인 리뷰** | 프로토타입·디자인 파일에 대한 사용성 피드백, 클릭맵, 마찰점 분석 | 엔진 활용 확장 예정 |
| **③ 제품 디스커버리** | 제품이 핵심 사용자 문제(JTBD)를 해결하는지 평가 | 엔진 활용 확장 예정 |
| **④ 전문가 리뷰** | 도메인 전문가 페르소나가 제품 기능과 시장 적합성을 평가 | 엔진 활용 확장 예정 |
| **⑤ 설문 조사** | 구조화된 설문을 페르소나 집단에 시뮬레이션. 세그먼트별 분석 | 엔진 활용 확장 예정 |
| **⑥ A/B 테스트** | 디자인/기능 변형을 비교하여 어떤 옵션이 더 효과적인지 판단 | 개발 예정 |
| **⑦ 퍼널 분석** | 전환 퍼널 각 단계별 이탈 원인과 병목 지점 분석 | 개발 예정 |
| **⑧ 광고 리서치** | 광고 컨셉, 포지셔닝, 메시지의 공감도를 페르소나 집단으로 테스트 | 개발 예정 |

> **핵심**: ①은 **Personica만 할 수 있다** — 경쟁사(Simfolk 포함)는 전부 "COMING SOON"이거나 미지원. ②~⑧은 경쟁사도 하지만, Personica는 **5-Layer 시뮬레이션 깊이**로 차별화된다.

**플랫폼 핵심 기능:**

| 기능 | 설명 |

## 3. 성장전략 (Scale-up)

### 3-1. 비즈니스 모델

**수익 모델:** 월간 구독 (SaaS)

| 플랜 | 가격 | 월간 테스트 | 페르소나 수/테스트 | 포함 기능 |
|---|---|---|---|---|
| **Free** | $0 | 1회 | 5명 | 모든 리서치 유형, 기본 리포트 |
| **Starter** | $49/월 | 10회 | 50명 | 오디언스 빌더, 세션 트랜스크립트, 사용성 점수 |
| **Pro** | $99/월 | 30회 | 200명 | Starter + 웹사이트 테스트(실제 브라우저), A/B 테스트, 퍼널 분석, 팀 워크스페이스 |
| **Scale** | $199/월 | 무제한 | 500명 | Pro + 반복 시뮬레이션(Day 1/7/30), 커스텀 리포트, 우선 지원 |

**원가 구조 (GPT-4.1 mini 기준 추정):**
- 200명 시뮬레이션당 LLM 비용: ~$5~10 / 브라우저 인프라: ~$5~10 / 서버: ~$2
- **매출 총이익률 78~90%** (플랜별)
- **손익분기: 유료 고객 15명** (Starter+Pro 혼합 기준)

**가격 앵커:** UserTesting.com에서 사람 5명 테스트 = $245+, 연간 계약 ~$46,000. Personica Pro에서 AI 200명 × 월 30회 = $99/월. LLM API 비용은 매년 80%씩 하락하는 추세로, 시간이 갈수록 수익률이 개선된다.

## 4. 팀 구성 (Team)

### 4-1. 대표자(팀) 구성 보유역량

| 역할 | 주요 역량 | 본 사업 기여 |
|---|---|---|
| **대표 / 엔지니어링 + 리서치** | AI 에이전트 개발, 감정 시뮬레이션 설계, 풀스택 개발, 인지심리학 논문 30+편 분석·적용 | 5-Layer 시뮬레이션 엔진 단독 설계·구현. 감정 모델 설계부터 대조 실험, 실제 리뷰 비교 검증까지 수행 |
| **[채용예정] 프론트엔드** | React, 데이터 시각화 | 웹 대시보드, 리포트 시각화, 오디언스 빌더 UI |
| **[채용예정] 그로스** | 커뮤니티 마케팅, 콘텐츠 | Product Hunt 런칭, 인디해커 커뮤니티 운영 |

**핵심 역량:** 대표가 직접 인지심리학·AI 시뮬레이션 분야의 최신 논문 30+편(PRISM, Generative Agents, Concordia, GenSim 등)을 분석하고, 이를 작동하는 시뮬레이션 엔진으로 구현했다. "AI가 성격과 감정에 따라 실제로 다르게 행동하도록 만드는" 기술적 난이도가 높은 영역에서 작동하는 엔진을 이미 확보했다.

**경쟁 우위(해자):**
1. **기술 깊이**: 5-Layer 인지심리학 아키텍처는 경쟁사의 프롬프트 엔지니어링과 구조적으로 다르다. 동일한 설계를 하려면 같은 수준의 논문 리서치와 통합 역량이 필요하다.
2. **실제 사용 능력**: 웹사이트 테스트 모드는 Personica의 독보적 기능이다. 디자인 파일을 "보고 추론"하는 것과 실제 브라우저에서 "사용하고 반응"하는 것은 결과 품질에서 근본적 차이가 있다.
3. **데이터 해자**: 시뮬레이션이 축적되면 업종별·세그먼트별 "사용성 벤치마크"가 쌓여, 시간이 지날수록 강해지는 데이터 경쟁 우위가 형성된다.

### 4-2. 중장기 사회적 가치 도입계획

- **사용자 리서치의 민주화:** 대기업만 접근 가능했던 종합 UX 리서치($46,000/년)를 $49~99/월로 낮춘다. 1인 창업자부터 소규모 팀까지, 누구나 출시 전에 다양한 사용자 관점에서 제품을 검증할 수 있다.
- **다양한 사용자 포용:** 비영어권, 고령층, 장애인 등 기존 테스트에서 빠지기 쉬운 사용자 유형을 AI 페르소나로 시뮬레이션한다. 더 포용적인 제품을 만드는 데 기여한다.
- **실패 비용 절감:** 출시 전에 치명적인 문제를 미리 발견하여, 스타트업의 리소스 낭비와 피봇 비용을 줄인다. 구축 전 수정이 구축 후 수정 대비 4~10배 저렴하다.
- **일자리 창출:** 사업 성장에 따라 직접 고용 확대 (2027년 3명 → 2028년 8명 목표)
