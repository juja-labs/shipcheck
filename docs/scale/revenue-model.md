### SOM과의 정합 확인

```
SOM (보수적): SAM $0.9B × 0.5% = $4.6M
SOM (기준):   SAM $1.7B × 1.0% = $16.5M

5년차 매출 추정: $3M~$6M

→ 보수적 SOM $4.6M과 5년차 $6M이 정합
→ 기준 SOM $16.5M은 7~10년차 또는 엔터프라이즈 가속 필요
→ 사업계획서에는 "5년 SOM $3M~$6M, 장기 $4.6M+" 으로 서술
```
