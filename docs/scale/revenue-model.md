```
매출 = 건수 × 건당 가격
```

| 티어 | 페르소나 수 | 스텝 | 건당 가격 | 타겟 세그먼트 | 용도 |

| 연간 사용 횟수 | 2~3회 (인디), 6~12회 (PM) | WTP 조사 기반 |
| Blended ARPU | 1년차 $600, 3년차 $2,000, 5년차 $3,000 | 세그먼트 Mix 변화 |

### 추정 결과

| 시점 | 누적 유료 고객 | Blended ARPU | **연 매출** | 주요 이벤트 |
|------|:----------:|:----------:|:--------:|-----------|
| **협약기간** (8개월) | 15~30건 (누적) | $149/건 | **$2K~$5K** | 알파→베타→오픈베타 |
| **1년차** | 200~400 | $600 | **$120K~$240K** | PH 런칭, 반복 사용 확인 |
| **2년차** | 500~1,000 | $1,200 | **$600K~$1.2M** | 모바일 확장, UX리서치 진입 |
| **3년차** | 800~1,500 | $2,000 | **$1.6M~$3M** | SMB 구독 + 미드마켓 |
| **5년차** | 1,000~2,000 | $3,000 | **$3M~$6M** | 엔터프라이즈 Pull |

### SOM과의 정합 확인

```
SOM (보수적): SAM $0.9B × 0.5% = $4.6M
SOM (기준):   SAM $1.7B × 1.0% = $16.5M

5년차 매출 추정: $3M~$6M

→ 보수적 SOM $4.6M과 5년차 $6M이 정합
→ 기준 SOM $16.5M은 7~10년차 또는 엔터프라이즈 가속 필요
→ 사업계획서에는 "5년 SOM $3M~$6M, 장기 $4.6M+" 으로 서술
```
