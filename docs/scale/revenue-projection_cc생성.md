### 5.1 매출원가 (COGS)

COGS에는 다음 항목이 포함된다:
- **LLM API 비용**: 페르소나 시뮬레이션의 모델 호출 비용 (경량 60% + 고성능 추론 40%)

- **리포트 생성 LLM**: 건당 1회 발생하는 분석 리포트 생성 비용
- **결제 수수료**: Stripe 2.9% + $0.30
- **환불 충당금**: 결제액의 3%

> 상세 건당 COGS는 `unit-economics.md` Section 2~4 참조

- **29년**: 외부 투자 없이 자체 수익($2,216K)으로 운영 + 현금 축적.

| 금액 | $3~5M (42~70억원) |
| 밸류에이션 근거 | 29년 $5M ARR × 10~15x = $50~75M |
| 목적 | 엔터프라이즈 세일즈 팀 구축, 마케팅 가속, 해외 거점 |
| 투자 후 팀 | 20~25명 |

> 시리즈 A는 필수가 아니라 **선택적 성장 투자**다.
> "이익을 내고 있는 상태에서 투자를 받는다"는 가장 강한 투자 유치 포지션이다.

## 9. 심사위원 관점 핵심 논리

1. **소수 정예 → 검증 후 확장**: 27년에 TIPS 투자를 확보하되 4명 체제를 유지하면서 제품-시장 적합성을 먼저 확인한다. 투자를 받자마자 인력을 급확장하는 것이 아니라, 매출이 증명된 뒤(28년) 확장하는 신중한 접근이다.

2. **현금이 한 번도 마이너스가 되지 않음**: TIPS $500K 유입 후 27년은 4명 운영으로 현금을 비축하고, 28년 팀 확장 기간에도 잔여 자금으로 커버한다. 자금 고갈 리스크가 없는 구조다.

3. **28년 성장 투자 후 29년 수익화**: 28년은 팀 확장으로 -9% 영업적자이지만, 이는 계획된 성장 투자다. 29년에 $5M 매출, 44% 영업이익률로 수익화에 성공하며, 이는 AI SaaS 업계에서 건강한 수준이다.

4. **채널 전환으로 ARPU 자연 상승**: 셀프서브(PAYG)에서 구독과 엔터프라이즈로 매출 비중이 이동하면서 ARPU가 $625 → $833 → $1,250으로 자연스럽게 올라간다. 가격 인상이 아닌 고객군 확장에 의한 ARPU 상승이다.

5. **COGS 지속 개선**: LLM API 가격 하락 추세와 프롬프트 캐싱 최적화로 매출원가율이 23% → 18%로 매년 개선. 매출총이익률 82%는 고수익 SaaS 수준이다.

6. **시리즈 A는 선택적**: 29년에 이익을 내고 있는 상태에서 투자를 받는 포지션. 생존이 아닌 가속을 위한 선택적 투자다.
