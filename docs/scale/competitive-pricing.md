### 가격 위치 (페르소나당, 로그 스케일)

```
$0.01 ─ SYMAR, Artificial Societies (무제한 월정액, 저품질)
$0.50 ─ Conjoint.ly (실제 사람, 설문 응답만)
$1    ─ Poll the People (실제 사람, 설문만)
$2~27 ─ Synthetic Users (텍스트 인터뷰, 최소 $10K/년)
$4.90 ─ ★ Personica ($4.90~$4.99/명) ← 여기
$5~50 ─ Ditto ($50~75K 연간에 묶임)
$8~10 ─ UserTesting (실제 사람, 세션 크레딧)
$22~98─ Respondent.io / User Interviews (실제 사람)
```

### Personica의 가격 경쟁력

1. **진입장벽 최저**: $49부터 건별 구매 (Starter 10명/30스텝). Synthetic Users($10K/년), Ditto($50K/년) 대비 압도적
2. **가치 대비 저가**: 동일 가격대 경쟁사는 텍스트 인터뷰만 제공. Personica는 Playwright 실사용 + 5-Layer 감정 시뮬레이션 + 행동 로그 + 리포트
3. **가격 인상 여지**: 현재 $4.90/명은 Synthetic Users 최저($2~27) 범위 내. Growth $149→$199~249, Scale $499→$599~699으로 올려도 경쟁력 유지
4. **인간 대비**: 동일 규모 인간 테스트의 1/4~1/330 비용
