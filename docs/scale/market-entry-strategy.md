## 2. 2차 시장: UX 리서치 소프트웨어

### 2.1 시장 구조

| 세그먼트 | 계약 규모/년 | 구매자 | 구매 방식 |
|---------|:----------:|--------|----------|
| 개인/프리랜서 | $0~$1,200 | UX 리서처 개인 | 셀프서브 프리미엄 |
| SMB | $1,200~$5,000 | PM, 디자이너 | 프리미엄 → 팀 업그레이드 |
| 미드마켓 | $5,000~$30,000 | UX팀 리드 | 챔피언 발견 → 팀 트라이얼 (1~3개월) |
| 엔터프라이즈 | $30,000~$180,000 | VP Product, Research Lead | RFP, 보안 심사 (3~12개월) |

참고: Maze 평균 SMB $2,840/년, 평균 엔터프라이즈 $72,768/년. UserTesting 엔터프라이즈 ~$148K/년.

### 2.2 성공한 신규 진입자의 GTM

| 기업 | 진입 방식 | 첫 100고객 | 핵심 전략 | 현재 |
|------|----------|-----------|----------|------|
| **Maze** | 프리미엄 (월 1회 무료) + SEO | PM/디자이너 셀프서브 | 템플릿 수백개 공동제작 → SEO. 연례 리서치 보고서. 전략적 투자자 (Atlassian, Zoom, HubSpot) | 6만 브랜드, 사용률 1위 (46%). $3M 매출 |
| **Dovetail** | MVP → 200유저 직접 아웃리치 → PLG | 리서처 네트워크 | 하나의 pain 해결 → 인접 페르소나 확산. SEO/콘텐츠 투자. 엔터프라이즈는 pull (알아서 찾아옴) | $630M 밸류에이션 |
| **Hotjar** | JS 1줄 설치 → 프리미엄 → 바이럴 | Bottom-up 바이럴 | 관대한 무료 티어 + 극도로 쉬운 온보딩 → 대규모 bottom-up 채택 | 3.3만 고객 → Contentsquare 인수 |
| **Lyssna** | 단일 기능 (사용성) → 확장 | 패널 구축 → 패널이 곧 채널 | 53만 패널이 자체 채널이자 해자. Figma/Zoom/Teams 통합 | 32만 유저 |

**공통 패턴:**
```
1. 하나의 날카로운 pain point 해결 (broad하지 않게)
2. 프리미엄으로 개인이 먼저 사용 (bottom-up)
3. 팀으로 확산 (챔피언이 조직 내 전파)
4. 엔터프라이즈는 pull — push 하지 않음
5. 콘텐츠/SEO가 장기 성장 엔진
```

## 3. 통합 로드맵

```
Phase 1a (0~6개월): 웹 제품 빌더 — 테스트베드
  ├─ 채널: Product Hunt, Show HN, BetaList, AppSumo
  ├─ 고객: 인디해커, 솔로빌더, 초기 PM
  ├─ 모드: 합성 서베이 (저마찰) + 제품 체험 (업셀)
  ├─ 가격: $49~$499/건 (PAYG, 복잡도별 티어)
  ├─ 목표: 유료 15~30건, PMF 신호
  └─ 전략적 의미: 사례 축적 + 검증 데이터 생성

Phase 1b (6~12개월): 모바일 앱 확장
  ├─ 연 25만+ 의미있는 앱 → WTP 13,850+
  ├─ Appium 기반 모바일 자동화 추가
  └─ 웹 검증 사례 → 모바일 적용

Phase 2 (12~18개월): UX 리서치 시장 진입
  ├─ 조건: 벤치마크 보고서 준비 완료
  ├─ 진입: Bottom-Up PLG (프리미엄)
  ├─ 채널: G2, ResearchOps, UXRConf, Figma 플러그인
  ├─ 포지셔닝: "탐색 단계 도구" (보완재)
  └─ ARPU: $2,400~$6,000/년 (SMB~미드마켓)

Phase 3 (18개월+): 엔터프라이즈 Pull
  ├─ Dovetail 패턴: 알아서 찾아옴
  ├─ SSO, 감사 로그, 데이터 거버넌스
  └─ ARPU: $30,000~$180,000/년
```

### SOM과의 정합

```
SOM 목표: SAM $0.9B~$1.7B 중 0.5% = $4.6M~$8.3M

현실적 경로:
  협약기간 $2K~$5K (유료 15~30건, Growth $149 기준)
  → 1년차 $120K~$240K (PAYG 반복)
  → 3년차 $1.6M~$3M (UX 리서치 + SMB)
  → 5년차 $3M~$6M (엔터프라이즈 ARPU 반영)

5년차 $6M은 SOM 보수적 $4.6M과 정합.
ARPU $3,000 × 2,000개사 = $6M
```
