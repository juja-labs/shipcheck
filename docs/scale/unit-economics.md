# Unit Economics — Personica

**Date**: 2026-03-22
**기준**: PAYG (Pay-As-You-Go) 건별 과금, 3개 시뮬레이션 모드
**모델 전략**: 경량 모델 (60%) + 고성능 추론 모델 (40%) 혼합
**원가 검증**: 30명 페르소나 시뮬레이션 검증 완료

## 1. 모델 전략 & 원가 구조

### 1.1 경량 6 : 고성능 추론 4 혼합

페르소나 리얼리티 엔진의 레이어별로 최적 모델 배정:

| 레이어 | 역할 | 모델 | 근거 |
|--------|------|------|------|
| Perceive (DOM 파싱) | 페이지 → 관찰 | **경량** | 구조화된 파싱, 창의성 불필요 |
| Act (행동 실행) | 제품 조작 명령 | **경량** | 단순 명령 변환 |
| Memory/Reflection | 기억 저장·회상 | **경량** | 패턴 매칭 + 요약 |
| Emotion (OCC-PAD-SDE) | 감정 평가·진화 | **고성능 추론** | 감정 뉘앙스가 핵심 |
| Decision (BDI-E) | 행동 결정 | **고성능 추론** | 동기·갈등·포기 시뮬레이션 |

토큰 비중: Perceive(DOM) + Act + Memory ≈ 60%, Emotion + Decision ≈ 40%

### 1.2 혼합 모델 원가

| 구분 | 토큰 비율 | Input 비용 | Output 비용 | 소계 |
|------|:--------:|:---------:|:----------:|:----:|
| 경량 60% | 141K in / 31K out | $0.113 | $0.155 | **$0.27** |
| 고성능 추론 40% | 94K in / 21K out | $0.283 | $0.312 | **$0.60** |
| **LLM 합계** | | | | **$0.87** |

> 30명 시뮬레이션 검증 기반 토큰 사용량 산출.
> Input은 프롬프트 캐싱 적용 후 실효 단가, Output은 공식 정가 기준으로 산출.
> 아래 Section 2~4의 모드별 원가는 실험 과금 데이터 기반 스텝당 단가($0.0115/스텝)로 산출하며, 이 이론 분해와 소폭 차이가 있을 수 있다.

#### 참고: LLM API 공식 가격 (2026-03 기준)

| 모델 | Input ($/MTok) | Output ($/MTok) | Batch (50% 할인) | 캐시 히트 (90% 할인) |
|------|:-------------:|:--------------:|:---------------:|:-----------------:|
| 경량 (Haiku 4.5) | $1.00 | $5.00 | $0.50 / $2.50 | $0.10 / — |
| 고성능 추론 (Sonnet 4.6) | $3.00 | $15.00 | $1.50 / $7.50 | $0.30 / — |

> 위 원가 산출은 프롬프트 캐싱 적용 후 실효 단가 기준.
> Batch API, 캐시 히트율 향상 등으로 추가 절감 가능.

## 2. 시뮬레이션 모드별 원가

페르소나 리얼리티 엔진은 3가지 모드로 제공되며, 모드별 원가 구조가 다르다.

### 2.1 모드 정의

| 모드 | 입력 | 제품 사용 | 턴 수/명 | 핵심 가치 |
|------|------|:--------:|:------:|----------|
| **합성 서베이** | 텍스트 질문 | ❌ | ~15 | 설문·인터뷰 시뮬레이션. 감정·인지 반응 포함 |
| **UI/UX 리뷰** | 텍스트 + 목업 이미지 (5~10장) | ❌ | ~25 | 디자인 목업에 대한 감정·인지·사용성 피드백 |
| **제품 체험** | 실제 URL | ✅ | 30~200 | 페르소나가 실제 제품을 직접 사용하며 행동·감정 시뮬레이션 |

> 모든 모드에 페르소나 리얼리티 엔진(감정·인지·의사결정 시뮬레이션)이 적용됨.
> 차이는 입력 방식과 턴 수이며, 시뮬레이션 품질은 동일.

### 2.2 모드별 COGS/페르소나

| 모드 | LLM/명 | 인프라/명 | **COGS/명** | 산출 근거 |
|------|:------:|:--------:|:----------:|----------|
| 합성 서베이 | $0.15 | $0.02 | **$0.17** | ~15턴, 텍스트 응답만 |
| UI/UX 리뷰 | $0.26 | $0.04 | **$0.30** | ~25턴, 이미지 토큰 추가 (화면당 ~2K토큰) |
| 제품 체험 (30스텝) | $0.35 | $0.04 | **$0.39** | 실제 제품 사용 |
| 제품 체험 (100스텝) | $1.15 | $0.10 | **$1.25** | 실제 제품 사용 |
| 제품 체험 (200스텝) | $2.30 | $0.19 | **$2.49** | 실제 제품 사용 |

> 위 COGS/명은 **시뮬레이션 비용만** 포함한다. **리포트 생성 LLM 비용**(건당 $0.30~$0.50)은 별도이며, Section 3.2와 4절의 COGS에 포함되어 있다.
>
> 제품 체험은 Memory Summarization으로 컨텍스트 윈도우를 관리하여 스텝당 비용이 급증하지 않도록 설계한다.
>
> **스텝 정의**: 제품 체험 모드에서 페르소나가 수행하는 행동 단위 1회(click, type, scroll, select 등).
> **스텝 기준 근거**: 내부 30명 실험에서 경량 웹 서비스(Tally) 대상 무제한 탐색 시 평균 73스텝이 소진되었다.
> 이를 기준으로 Starter 30스텝은 핵심 시나리오 1개 집중 검증, Growth 100스텝은 중간 복잡도 서비스의 핵심 플로우 커버,
> Scale 200스텝은 복합 플랫폼의 다중 플로우 탐색을 목표로 설정했다.


### 2.3 페르소나 규모별 분석 범위

| 페르소나 수 | 세그먼트 커버 | 의미 |
|:---------:|:----------:|------|
| 10명 | 2~3개 | 방향성만 확인 |
| 30명 | 9개 | 세그먼트별 경향 확인 가능 |
| 100명 | 세그먼트별 10명 | 세그먼트 내 일관성까지 검증 |

> **근거**:
> - **자체 30명 실험**: Big Five 성격 모델 기반으로 9개 세그먼트를 자동 생성하여 시뮬레이션.
>   세그먼트 간 유의미한 행동 분화(이탈 지점, 감정 반응, 기능 사용 패턴 차이)를 확인함.
> - **Nielsen & Landauer (1993, ACM INTERCHI)**: 사용성 테스트에서 5명이면 이슈의 약 85%를 발견. 10명은 방향성 확인에 충분.

- Delve AI $99/월 대비 → **월정액 없이 필요할 때만 구매 가능**
- 페르소나 리얼리티 엔진의 감정·인지·의사결정 시뮬레이션을 포함한 PAYG 포지션

- 디자인/프로토타입 검토를 **월정액이 아닌 1회성 PAYG**로 제공
- 목업 이미지 기반이지만 페르소나 리얼리티 엔진 해석(감정·인지·사용성)을 포함

**제품 체험 ($149/30명 = $4.97/명)**:
- **카테고리 유일** — AI 페르소나가 실제 제품을 직접 사용하는 시뮬레이션을 제공하는 경쟁 제품 없음
- 인간 리크루팅 플랫폼의 30세션 비용 대비 **최소 7배 이상 저렴** (아래 ③ 참조)

## 4. 건당 손익

> **이 절을 읽는 기준**
> - 아래 값은 **현재 기준 건당 공헌이익**이다.
> - 월 고정비(인건비, 상시 호스팅, DB, 모니터링, 업무용 소프트웨어)는 별도 문서에서 관리한다.
> - 현재 단계에서는 대표 주도의 자연 유입을 전제로 하므로, **고객 획득 비용은 건당 손익에 별도 반영하지 않는다**.
> - 초기 단계의 고객지원·온보딩 응대 역시 팀 인건비가 흡수한다고 본다.
> - **리포트 생성 LLM 비용**은 시뮬레이션 종료 후 건당 1회 발생하는 추가 원가다.
> - **환불 충당금**은 초기 운영 리스크를 보수적으로 반영해 결제액의 3%로 둔다.
